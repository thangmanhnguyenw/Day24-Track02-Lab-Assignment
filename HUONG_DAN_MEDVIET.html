MedViet là startup y tế Việt Nam xử lý hồ sơ bệnh nhân để train mô hình chẩn đoán. Công ty cần đạt chuẩn NĐ13/2023 và ISO 27001 trước khi ký hợp đồng đối tác.

  • Kiểm soát truy cập RBAC (Casbin) + ABAC (OPA)
  • Mã hóa envelope encryption (AES-256-GCM)
  • Validation chất lượng dữ liệu (Great Expectations)
  • Security scanning (git-secrets, Bandit, TruffleHog)
  • Compliance checklist NĐ13

4.2 Anonymization — src/pii/anonymizer.py

3 strategies trong anonymize_text():

  • replace — Faker vi_VN (tên, email, CCCD, SĐT giả)
  • mask — ký tự * che toàn bộ entity
  • hash — SHA-256 one-way hash

anonymize_dataframe() xử lý từng cột:

CộtCách xử lý

ngay_sinhGeneralize — chỉ giữ năm sinh
patient_id, benh, ket_qua_xet_nghiemGiữ nguyên

4.3 RBAC — src/access/rbac.py + policy.csv

get_current_user() parse Authorization: Bearer token-xxx → HTTP 401 nếu thiếu/sai token.

require_permission() gọi enforcer.enforce(username, resource, action) — dùng username (alice, bob...) không phải role, vì Casbin matcher dùng g(user, role).

Đã bổ sung policy cho admin: training_data:read, aggregated_metrics:read và ml_engineer: aggregated_metrics:read.

Phần 4 — Encryption Envelope (30 phút)

flowchart TB subgraph KEK_Layer["Master Key Layer"] MK[.vault_key
KEK 256-bit] end subgraph DEK_Layer["Data Key Layer"] DEK[Random DEK 32 bytes] EDEK[encrypted_dek = nonce + AESGCM_KEK encrypt DEK] end subgraph DATA_Layer["Data Layer"] PT[Plaintext string] CT[ciphertext = nonce + AESGCM_DEK encrypt data] end MK --> EDEK DEK --> EDEK DEK --> CT PT --> CT CT --> PAYLOAD["{encrypted_dek, ciphertext, algorithm}"]

Luồng encrypt / decrypt

  1. encrypt_data(): Generate DEK mới → encrypt data bằng DEK → encrypt DEK bằng KEK → xóa plaintext DEK khỏi memory → trả payload base64
  2. decrypt_data(): Decode base64 → decrypt DEK bằng KEK → decrypt data bằng DEK → trả plaintext
  3. encrypt_column(): Áp dụng encrypt_data cho từng cell trong cột DataFrame

Test trong Python shell

cd medviet-governance
python
>>> from src.encryption.vault import SimpleVault
>>> vault = SimpleVault()
>>> original = "Nguyen Van A - CCCD: 012345678901"
>>> encrypted = vault.encrypt_data(original)
>>> decrypted = vault.decrypt_data(encrypted)
>>> assert decrypted == original
